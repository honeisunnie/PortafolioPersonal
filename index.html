Modelo de Predicción de Cáncer de Mama

2023 - Proyecto de Ciencia de Datos

Machine Learning - Clasificación Binaria

Desarrollo de un modelo predictivo utilizando regresión logística para la detección temprana de cáncer de mama basado en características clínicas.

  • Análisis exploratorio de datos (EDA) para comprender las características del dataset
  • Preprocesamiento de datos: manejo de valores faltantes y normalización
  • Entrenamiento y evaluación de modelo de regresión logística
  • Métricas de evaluación: precisión, recall, F1-score y matriz de confusión
  • Visualización de resultados y características más relevantes

Tecnologías utilizadas: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

Resultados: Modelo con precisión del 96% en la detección de casos positivos.

Sistema de Diagnóstico de Enfermedades Dermatológicas

2023 - Proyecto de Machine Learning

Aprendizaje Supervisado - Clasificación Multiclase

Implementación de un clasificador para predecir enfermedades de la piel basado en características clínicas e imágenes dermatológicas.

  • Procesamiento de imágenes médicas y extracción de características
  • Implementación de diferentes algoritmos de clasificación
  • Optimización de hiperparámetros para mejorar el rendimiento
  • Desarrollo de una interfaz simple para realizar predicciones
  • Análisis comparativo entre diferentes enfoques de modelado

Tecnologías utilizadas: Python, OpenCV, TensorFlow/Keras, Scikit-learn, Streamlit

Resultados: Precisión del 89% en la clasificación de 7 tipos diferentes de enfermedades dermatológicas.
